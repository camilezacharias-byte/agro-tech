# Produtividade com Consciência 🌾🔄

> Plataforma interativa de inteligência agrícola que demonstra a sinergia entre a alta produtividade no campo e a preservação socioambiental.

Este projeto foi desenvolvido como um portal interativo (One-Page) para conectar conceitos modernos de **Agrotecnologia (AgTech)** com práticas reais de sustentabilidade no campo, trazendo dados dinâmicos e desafios técnicos para o usuário.

---

## 🚀 Funcionalidades Principais

* **Dashboard de Indicadores Dinâmicos:** Um gráfico interativo que permite ao usuário alternar em tempo real entre diferentes métricas (Visão Geral, Eficiência Hídrica e Pegada de Carbono).
* **Central de Desafios (Quiz):** Um simulador de tomada de decisões onde o usuário testa seus conhecimentos sobre o manejo sustentável e recebe pontuação ao final.
* **Interface Adaptativa (Dark Mode):** Suporte completo para troca de temas (claro e escuro) focado em acessibilidade e conforto visual.
* **Design Responsivo:** Interface otimizada para funcionar perfeitamente em computadores, tablets e smartphones.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento web front-end nativo, mantendo a estrutura modularizada e limpa:

* [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML) - Estruturação semântica do portal.
* [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS) - Estilização moderna baseada em variáveis de escopo (`:root`) e Flexbox/Grid.
* [JavaScript (ES6)](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) - Lógica do quiz dinâmico e manipulação de estados do tema.
* [Chart.js](https://www.chartjs.org/) - Biblioteca externa utilizada para a renderização e animação dos gráficos de dados.
